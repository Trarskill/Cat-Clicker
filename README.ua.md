# 😺 An Epic Tail: Cat Clicker v0.1n


### 🖥️ Платформа:
**PC - Windows \ Mobile - Android/iOS**

--- 

### 📘 Опис:

"An Epic Tail: Cat Clicker" — це 2D Idle-Clicker RPG, розроблена на базі рушія Godot Engine.
Гра пропонує повноцінний ігровий процес класичного клікера, в гру інтегрований повноціна механіка менеджменту ресурсі, та функціонуючий магазин. 
Гра має розвинену рівневу систему прогресії (XP), де при отриманні кожного нового рівння гравець отримує унікальну нагороду, також реалізовано продуману систему "Ендгейму" на максимальному (100-му) рівні, яка перетворює нескінченний геймплей на циклічне джерело винагороджень без лімітів.

--- 

### 🖼️ Художній опис гри:

Допоможіть звичайному сільському котику стати хоробрим воїном лісу. Він не зміг вступити до лицарів, але має велике бажання тренуватися. Зробіть його дедалі сильнішим, тренуючи під деревом удари по маківарі, а після тренувань вирушайте в ліс, на поля та в гори, щоб битися з ворогами. Згодом котик стане сильним, і у нього з'явиться нова мета — перемогти дракона в замку з монстрами та визволити кицю.

### 🚀 Запуск:

1. Відкриваємо Godot Engine 4.3+
2. Натискаємо на import > знаходимо папку гри "An Epic Tail: Cat Clicker"
3. Відкриваємо проект
4. Якщо потрібно імпортуємо в проект "Assets" (тільки через рушій)
5. Run Project (F5)

- CoinBar ( елемент хедера, панель з кількістю монет, три валюти: монети, рідкі монети, геми ) 
- SettingsButton

- Header ( верхня панель )
- Background ( задня панель, красива панель яка згодом може змінюватися на іншу )

- Dashboard ( основний екран клікера, початковий екран )
- ShopPopup ( три вікна з персонажами торговцями та їх асортиментом товару )
- MapPopup ( карта світу гри, з локаціями(..в майбутньому) )
- BattlePlate ( вікно бойової частини ховаються lowbar та додаються кнопки бійки)
- BootScreen ( окно при заході в гру та зміні режиму(..в майбутньому) )

### 📈 Pеалізовано в проекті:

[✔] version 0.1n:

1. **Структура клікер:** Простір для клікання та ігровий HUD.
2. **Система прогресії:** Отримання досвіду (XP), підвищення рівнів до максимального (100-го) та унікальна система "Ендгейму" з циклічним отриманням нагород (монети та геми) за продовження гри на максимальному рівні.
3. **Ігрова економіка та Магазин:** Система торговців із категоріями товарів, що функціонує на базі кількох валют.
4. **Динамічний інвентар:** Сітка з предметів, яка звідображує поточні наявні предмети гравця.
5. **Функціональні предмети з унікальними властивостями:** Кожен предмет в грі має унікальну логіку активації, перевірку умов використання, таймери дії інше.
6. **Інтерактивні UI-елементи:** Інформаційні вікна предметів з кнопками контекстної дії, таймерами автозакриття та візуальним попередженням при помилці використання.
7. **Візуальний відгук (Juiciness):** Спливаючий текст (Floating Text) для сповіщень про нагороди та нові рівні, а також плавні Tween-анімації для переміщення камери та появи вікон.
8. **Глобальний менеджмент стану:** Система збереження та завантаження ігрового прогресу (рівні, валюта, інвентар) у локальний JSON-файл.
