# 😺 Cat Clicker v0.2 beta


### 🖥️ Платформа:
**PC - Windows(вертикально) \ Mobile - Android**

--- 

### 📘 Опис:

"Cat Clicker" — це 2D Idle-Clicker RPG, розроблена на базі рушія Godot Engine.
Гра пропонує повноцінний ігровий процес класичного клікера, в який інтегрована повноцінна механіка менеджменту ресурсів та функціонуючий магазин. 
Гра має розвинену систему рівневої прогресії (XP), де при отриманні кожного нового рівня гравець отримує унікальну нагороду. Також реалізовано продуману систему "Ендгейму" на максимальному (100-му) рівні, яка перетворює нескінченний геймплей на циклічне джерело винагород без лімітів.

--- 

### 🖼️ Художній опис гри:

Допоможіть звичайному сільському котику стати хоробрим воїном лісу. Він не зміг вступити до лицарів, але має велике бажання тренуватися. Зробіть його дедалі сильнішим, тренуючи під деревом удари по маківарі, а після тренувань покращуйте зброю та навички котика, щоб стати ще сильнішим. Згодом котик стане могутнім, і у нього з'явиться нова мета — перемогти дракона.

### 🛠️ Технології: 
- Godot Engine 4.3
- GDScript

---

### 🚀 Запуск:

1. Відкриваємо Godot Engine 4.3+
2. Натискаємо на import > знаходимо папку гри "Cat Clicker"
3. Відкриваємо проект
4. Якщо потрібно імпортуємо в проект "Assets" (тільки через рушій)
5. Run Project (F5)

### 👨‍💻 (Старий) План покращення:

1. Основана частина гри - створення інтерфейсу, у вигляді компонентів: 
	
- CustomButton ( різні, красиві, кастомні кнопки ) 
- LowBar ( головна нижня панель навігації: кнопка магазину, кнопка карти мапи, прогресбар рівня )
- LevelBar ( прогрес бар у вигляді кнопки з надписом кількістю XP та звеху поточного рівня персонажа)
- CoinBar ( елемент хедера, панель з кількістю монет, три валюти: монети, рідкі монети, геми ) 
- SettingsButton
- SettingBar ( налаштування з зміною функціональності(..в майбутньому) )
- Header ( верхня панель )
- Background ( задня панель, красива панель яка згодом може змінюватися на іншу )
- Character ( кіт, у можливості додавання анімацій та зміни вигляду(..в майбутньому) )
- Dummy ( перший але не зовсім ворог )
- Dashboard ( основний екран клікера, початковий екран )
- ShopPopup ( три вікна з персонажами торговцями та їх асортиментом товару )
- MapPopup ( карта світу гри, з локаціями(..в майбутньому) )
- BattlePlate ( вікно бойової частини ховаються lowbar та додаються кнопки бійки)
- BootScreen ( окно при заході в гру та зміні режиму(..в майбутньому) )

2. Закріплення коду та інтерфейсу клікерної частини.
3. Перша вкладка - створення магазину з трьома вкладками, пасивне отримання валюти, яка функціонує в магазині.
4. Друга вкладка - створення карти з різними локаціями, які відкриваються при досягненні певного рівня досвіду, а також додавання квест таблиці.
5. Бойовий Дешборд - створення бойової частини, вона відрізняється від клікерної тим, що там є окремі кнопки для атак та захисту, а також окремий інтерфейс для відображення здоров'я та інших параметрів персонажа та ворога, lowbar clicker зникає з'являється новий lowbar battle.
6. Додання анімацій: персонажу, ворогам(1-2 ворога).
7. Оновлення магазину, оновлення функціоналу гри.
8. Додавання візувлу для гри: Звуки, Більше анімацій, i18n інше..

### 📈 Pеалізовано в проекті:

[✔] version 0.1n:

1. **Структура клікера:** Простір для клікання та ігровий HUD.
2. **Система прогресії:** Отримання досвіду (XP), підвищення рівнів до максимального (100-го) та унікальна система "Ендгейму" з циклічним отриманням нагород (скринь) за продовження гри на максимальному рівні.
3. **Ігрова економіка та Магазин:** Система торговців із категоріями товарів, що функціонує на базі кількох валют.
4. **Динамічний інвентар:** Сітка предметів, яка відображає поточні наявні предмети гравця.
5. **Функціональні предмети з унікальними властивостями:** Кожен предмет у грі має унікальну логіку активації, перевірку умов використання, таймери дії тощо.
6. **Інтерактивні UI-елементи:** Інформаційні вікна предметів з кнопками контекстної дії, таймерами автозакриття та візуальним попередженням при помилці використання.
7. **Візуальний відгук (Juiciness):** Спливаючий текст (Floating Text) для сповіщень про нагороди та нові рівні, а також плавні Tween-анімації для переміщення камери та появи вікон.
8. **Глобальний менеджмент стану:** Система збереження та завантаження ігрового прогресу (рівні, валюта, інвентар) у локальний JSON-файл.
